🏷️ IDs y Classes

id es único. class se puede repetir.

box + red-box
box + blue-box

Este título tiene id="title" (arriba). En CSS se selecciona con #title. Las clases se seleccionan con .box.

🧩 HTML Entities

Se usan para escribir caracteres reservados como < > &.

  • Less than: &lt; → <
  • Greater than: &gt; → >
  • Ampersand: &amp; → &
  • Yen: &yen; → ¥
